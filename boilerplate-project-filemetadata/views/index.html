Upload a File

Example Output:

{
  "name": "example.jpg",
  "type": "image/jpeg",
  "size": 12345
}

API Documentation

Send a POST request to /api/fileanalyse with a file in the "upfile" field.

The response will be a JSON object containing the file's name, type, and size in bytes.
